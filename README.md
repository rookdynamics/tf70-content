# tf70-content

Editorial content repository for Task Force 70.

Audience: Springfield Armory Magazine, 2A supporters and advocates.
Voice: 60% Col. Jeff Cooper / 40% Law Enforcement Professional.

## Structure

| Directory | Purpose |
|-----------|---------|
| `drafts/` | In-progress pieces |
| `published/` | Final, published pieces |
| `archive/` | Retired or superseded content |
| `assets/` | Images, logos, supporting files |

## Naming convention

```
[YYYY-MM]_[slug].md
```

Example: `2026-05_the-obligation-of-ownership.md`
